Port Mapping:
	docker run -p 9000:8080 {Image-Id}

here 
	9000 is new or custom port
	8080 is default port

Check it in website
	http://localhost:9000/

	By clicking this niginx application will be UP


Rename the Container Name ?
	docker run --name Pradeep -p 9000:8080 da937936b3c6


Types Of Model Executing The Container 
	
	1)	Attach Mode : The server is running in fore ground (if i close the server docker will be down)
			docker run --name Pradeep -p 8080 {Image-ID}

	2)	De-Attach Mode : The server is running in back ground
			docker run --name Pradeep -p 8080 -d {Image-ID}
