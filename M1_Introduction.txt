To Create Docker Container from docker image ?
	docker run {container-id}


To Create Docker Container from docker image Port Mapping ?
	docker run -p 27017:8080 {Image Id}

Remove Container ?
	docker rm {container-id}

Note : You can't remove the active container and you have to stop and remove

Remove the Acive Container ?
	docker rm  -f {container-id}



Remove All Container ?
	docker rm $(docker ps -a -q)
		where $ = Variable


Remove Image ?
	docker rmi {Image-ID}
